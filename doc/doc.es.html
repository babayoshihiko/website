Fink - Documentacion

Este es una coleccion de varios documentos escritos para el Fink. Algunos de ellos tambien pueden ser utiles para la gente que usa Mac OS X o Darwin sin Fink y que deseen aprender a hacer "puertos" de software de Unix.

Documentecion del Usuario

La documenteacion actual del Fink incluye:

  • Guia del Usuario del Fink - esto cubre la instalacion del propio Fink, la instalacion de pauqtes, y actualizaciones del Fink. Contiene instrucciones tanto para la version fuente como para la binaria. Trabajo en Proceso!
  • Ejecutando X11 en Mac OS X y Darwin - cubre conceptos como la instalacion y ejecucion (tambien dedicado a los usuarios de Darwin y Mac OS X)

Un grupo de documentos que son mas completos, pero ligeramente desactualizados y no frecuentemente actualizados son los que siguen:

  • Instalacion y Actualizacion - cubre la instalacion de Fink asi como las actualizaciones
  • Uso - como usar Fink e instalar programas
  • Fink ReadMe - El archivo Leeme de la distribucion fuente
  • Acceso a CVS - Como acccesar aldeposito CVS de Fink para obtener las ultimas version de codigo fuente de los paquetes, aun entre liberaciones del Fink

Documentecion de Desarrolladores

  • UUsando Fink: Un "Como hacer" para los desarrolladores (un pdf de 2 Mbs) - transparencias e la presentacion de la ConferenciasO'Reilly Mac OS X (Disponible tambien como archivo de PowerPoint )
  • Tips para hacer "puerto" - notas para hacer "puertos" de aplicaciones de Unix a Darwin
  • Manual de Paquetes - como crear y mantener paquetes de Fink
